Mostra elements ocults

Especifica com es mostren els elements de qualificació ocults. Si seleccioneu No mostris cap element ocult, resten completament ocults. Si seleccioneu Mostra tots els elements ocults, la fila de l'element de qualificació ocult es mostra en gris amb la qualificació completament oculta. Si seleccioneu Mostra només els elements "oculta fins", els elements de qualificació que tinguin definida una data "oculta fins" es mostren en gris amb les qualificacions completament ocultes fins la data definida, i després d'aquesta data es mostrarà l'element complet.
